# Overview
Shortcuts for existing filters for SWUDeck. Some keywords are already short enough like "hp" or "cost". But others could benefit from shortening them to 1-3 characters.

# Requirements
REQ-1 aspects:
  REQ-1.1: "c" for aspect (as in color)
  REQ-1.2: filter on values as well: b for Vigilance (as in blue), g for Command (as in green), r for Aggression (as in red), y for Cunning (as in yellow), w for Heroism (as in white), k for Villainy (as in black)
  REQ-1.3: value shortcuts can be combined to search for multiple aspects, for example c:bk is the same as "aspect:blue aspect:villainy"; c=bk is the same as c:bk;
REQ-2: "t" for text
REQ-3: "p" for power
REQ-4: hp is already short enough
REQ-5: "tr" for trait
REQ-6: "up" for upgrade power
REQ-7: "uhp" for upgrade hp
REQ-8: "r" for rarity
REQ-9: "a" for arena
