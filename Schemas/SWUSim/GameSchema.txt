# ── CardID Format Rule ────────────────────────────────────────────────────────
# Every CardID stored in any zone must use the SET_NNN format:
#   {set abbreviation (2–5 uppercase letters)} + "_" + {card number zero-padded to 3 digits}
#
# Examples:  SOR_014   SHD_154   JTL_095   LOF_102
#
# This format is the primary key across all game state (zone entries, DQ variables,
# mzIDs, generated dictionaries, and the deck JSON import format).
#
# For stat reporting to SWUDeck/SWUStats, call GetCardUUID(cardId) to resolve
# SET_NNN → documentId (Strapi UUID). See GeneratedCardDictionaries.php.
# ─────────────────────────────────────────────────────────────────────────────

# Discard ──────────────────────────────────────────────────────────────────────
# Defeated units, played events, and discarded cards. Public and face-up.
# Order does not matter and may be rearranged freely.
#
# Fields:
#   From     — origin zone when the card entered discard:
#                "HAND"      — discarded from hand (regroup discard, card effect)
#                "DECK"      — milled from top of deck
#                "PLAY"      — left the arena (unit defeated, event resolved)
#                "RESOURCES" — played from resource zone (Smuggle / similar)
#   Turn     — TurnNumber when the card was discarded; used for "discarded this phase/round" checks
#   Modifier — optional tag on the discard entry:
#                "TTFREE"  — may be played for free this turn (this player)
#                "OTTFREE" — may be played for free this turn (opponent)
#                ""        — no modifier (default)

# Resources ────────────────────────────────────────────────────────────────────

# Leader ───────────────────────────────────────────────────────────────────────
# The leader card in the base zone. Always stays in this zone for record-keeping;
# deployment state is tracked by Deployed + DeployedUniqueID.
#
# Fields:
#   EpicActionUsed  — true after the Epic Action (deploy) has been triggered once per game
#   Ready           — false when the leader side is exhausted (for leader-side action abilities)
#   Deployed        — true when the leader unit side is currently on the field
#   DeployedUniqueID — UniqueID of the corresponding GroundArena/SpaceArena entry (0 if not deployed)
#   Damage          — damage on the leader UNIT side while deployed (leader side has no HP)
#   TurnEffects     — temporary effects applied this phase/round/attack
#   Counters        — named counter map (e.g. epic action counter display)
#
# When Deployed=true, a matching entry in GroundArena (or SpaceArena) represents the
# Leader Unit side. On defeat of that entry, Deployed is reset to false and the leader
# returns to the base zone exhausted (Ready=false), per CR 3.4.5.
